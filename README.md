# Multifrequency Inter-Scale Paper

Manuscript for "Signal-Preserving Machine Learning for CMB Foreground Reconstruction: Frequency-Difference and Inter-Scale Approaches"

## Contents

- `journal_manuscript.tex` - Main LaTeX manuscript
- `references.bib` - Bibliography
- `aas_macros.sty` - AAS journal macros
- `Figures/` - All figures for the manuscript
- `MANUSCRIPT_GUIDE.md` - Guide for manuscript preparation
- `methodology_polished.md` - Methodology documentation

## Compilation

To compile the manuscript:

```bash
pdflatex journal_manuscript.tex
bibtex journal_manuscript
pdflatex journal_manuscript.tex
pdflatex journal_manuscript.tex
```
